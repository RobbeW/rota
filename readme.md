# Rota 

Interactieve **1v1 Rota** (bordspel uit de oudheid) met **mini-Latijn** in een Duolingo-achtige flow. Gemaakt voor klasgebruik (laptop, tablet, telefoon).

## Speel de game
https://robbew.github.io/rota/

## Wat is dit?
- **Rota**: 2 spelers, elk **3 stukken**. Eerst **plaatsen**, daarna **schuiven**.
- **Doel**: **drie op een rij door het midden**.
- Tijdens het spelen oefen je een kleine set **basiszinnen Latijn** via korte micro-oefeningen.

## Kernregels (kort)
1. **Plaatsen**: om beurten plaats je één stuk op een leeg punt tot iedereen 3 stukken heeft.  
2. **Schuiven**: verplaats om beurten één stuk naar een **direct verbonden** leeg punt (**exact één stap**).  
3. **Winnen**: maak **drie op een rij door het midden**.

## Hulp & printversie
- Klik op **?** (rechtsonder) voor speluitleg en zinnenlijst.
- In de help-modal: **“Maak printbare versie”** genereert een **PDF** met bord + namen + zinnen om offline te spelen.
